# Homelab

## Overview
This repository contains documentation, configuration backups, Docker Compose files, diagrams, and recovery procedures for the homelab.

## Main Systems
- Dell OptiPlex 7060
- Raspberry Pi 4
- Verizon CR1000B router
- External 8TB media drive
- SMB backup share on desktop PC

## Key Services
- Jellyfin
- Immich
- Vaultwarden
- Homepage
- Uptime Kuma
- Caddy
- qBittorrent + Gluetun
- Sonarr / Radarr / Bazarr / Prowlarr / Seerr
- Pi-hole + Unbound

## Documentation Sections
- inventory/ → hardware, ports, services, domains, IPs
- compose/ → Docker Compose files
- configs/ → important configs
- runbooks/ → restore and troubleshooting guides
- diagrams/ → network and service diagrams
- backups/ → backup scripts and exports
